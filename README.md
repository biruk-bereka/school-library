### Key Features <a name="key-features"></a>

- Person class created
- Student class created
- Teacher class created
- Base Decorator created
- Person class decorated with Decorator
- Create a class Classroom
- Create association between classroom and student 
- Create a class Book 
- Create a class Rental
- Create association between book and rental 
- Create asssociaiton between person and rental

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Git SCM
- Visual Studio Code
- Live Server extension

### Setup

Clone this repository to your desired folder:

```
git clone https://github.com/biruk-bereka/school-library.git
```

Or download the ZIP folder.

### Install

Once you've cloned the repository simply run:

```
cd school-library

```

And this should open the directory with all the files from the repository on the command line, to open it in your explorer if you're using windows you can run:

```
explorer .
```

### Usage

To run the project, if you are on the school-library folder path on your terminal you can execute the following command:

```
code .
```

This will open Visual Studio Code on the folder you're at.

open terminal and run

```
 irb .\person.rb
 irb .\student.rb
 irb .\teacher.rb
```

### Run tests

There are no tests configured for this project.

### Deployment

This project is not deployed
